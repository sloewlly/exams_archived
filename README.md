## Greeting folks!

Hi fellow VGU-ers! I'm Andy, and this is my archive for the "Programming - 2" course. By the time this repo goes live, you’re likely tackling this subject in your sophomore year, unless, of course, "Programming - 1" threw you for a loop first. Either way, you know the drill: you need solid material to study and practice with so you can survive the massive final exam. This course is a massive deal, particularly if you’ve got your sights set on that FRA-UAS scholarship to spend next year's summer semester studying in Germany! So, seriously folks, **DO NOT FAIL** this!

This repo should contain:
* Coursework (of CSE2024)
* Practise exams (Archived by Linh "Siga")
* Solutions to the questions (Remember to use it responsibly!)

To wrap things up, best of luck to everyone reading this! Huge thanks for dropping by.

~ Best wishes, Andy
